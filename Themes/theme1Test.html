Introduction

In this assignment we carry out an analysis of a dataset containing information of about 40 thousand real estates sold in Iceland from 2008 to 2018. The origin of the data can be traced to the National Registry of Iceland which performs an evaluation of properties worth every year. The data is from 2019 so the current property worth will be referred to simply as property worth.

The first part will explore the data visually. The second part will predict properties worth using the lasso and boosting methods. The third part will predict whether a property is old or new using QDA, logistic regression and random forest. Stick around to see if we cracked the bonus question…

The original dataset was very high dimensional. Irrelevant variables and observations were excluded. Focus will be on data concerning single-family houses and apartment building located in the Capital Region of Iceland. The resulting dataset contains approximately 35 thousand observations and 17 variables regarding properties worth, municipality, age of construction, property type and property area among other.

Part II: Predicting property worth

In this part we will predict sale worths using the lasso and boosting methods. We will start by doing a factor analysis of the dataset to get a better idea of its nature.

Factor analysis on the data set as a whole obtained 25 dimensions all with eigenvalues higher than one, indicating that the dimension accounts for more variance than one of the original variables. These 25 dimensions only collectively explained around 66% of data variance. Neighbourhood had no correlation to any dimensions so it will be removed for further analysis, as will worth per square meter since it has a high correlation to other predictors.

Figure 9 shows the relative and cumulative variance explained by the factor analysis dimensions. Removing the two variables reduced the dimensions from 25 to seven and only decreased the cumulative variance explained by around 2%.

Figure 10 shows how much data variability the variables explain. Clearly, municipality explains the most variability compared to the other variables.

Here we can see how the variables influence each dimension. The plot indicates that we should look at each municipality seperately and aligns with previously seen information (see Figure 10). Additionally, it seems like the number of showers and baths a property has might work together, perhaps as number of master bathrooms. Sales year seems like a strong variable on it’s own.

Only the first seven dimensions had eigenvalues above one. Therefore we might assume that splitting the municipality variable into its factors would only make two of it’s factors strong predictors. All variables included in the first dimension, except for property pricing and property type, might indicate a single factor belonging to property size.

Figure 12 shows the 19 predictors that Lasso deemed influental regarding property worth. The bottom seven variables have a negative relationship with property worth while the remaining have a positive relationship. The variables Seltjarnarnes, number of floors, year the property was bought, Garðabær, Hafnarfjörður and number of garages are the most influential according to the lasso.

The model explained around 76% of property worth variance (\(R^2\)). With training error around 36.95 m.ISK and test error around 36.84 m.ISK.

In alignment with the factor analysis, the municipality regions do seem influential for property worth as seperate predictors. Additionally, number of toilets and bathtubs seem to have similar effect on property worth and might be considered as a singular predictor as noted earlier.

The main issue with Lasso are correlated/confounding variables. As seen earlier (see Figure 10) square meters (size of property) came in second when it comes to explaining variance in the data. In figure 12 this variable (size) is included but its coefficient has been reduced almost to zero. Meanwhile, all the variables from the first dimension of the factor analysis appear to also be included here. Lasso is blind to the influence of square meter on property worth (due to high correlation) and it is important to keep that in mind and not exclude the effect that square meter has on property worth.

In Figure 13 we see the relationship between lassos’ predicted values and observed values. The right graph includes all values in the data set and we appear to have an extreme value at the top of the plot. The left graph is a close-up of all the values excluding the extreme value. The relationship seems stronger for properties of lower worth but starts to gradually worsen as the worth goes up.

Boosting

We ran cross-validation on multiple combinations of trees. Tuning the parameters resulted in a model with 500 boosting iterations (n.trees = 500), a max tree depth of ten (interaction depth = 10), and a \(\lambda\) of 0.1. The minimum terminal node size was kept constant at 10.

In figure 14 we see a plot of the RMSE via cross-validation according to various measures of tree size and shrinkage value. We can see here that using 500 boosting iterations, with \(\lambda\) of 0.1 and a max tree depth of 10 results in the lowest RMSE out of all the tree combinations that were tested.
